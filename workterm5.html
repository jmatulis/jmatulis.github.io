Experience

Daily Operations

My day-to-day work primarily involved provisioning and deprovisioning physical laptops and virtual machines for new hires, hardware replacements, and internal team transitions. I processed returned laptops, performed quality checks, and ensured devices were compliant with internal policies before redeployment. I also remotely installed applications, configured user accounts, and assisted users with setup through remote desktop support.

Ticket Queue Management

  • Managed a technical support queue for software and hardware requests across the organization.
  • Provisioned laptops with department-specific configurations and ensured compliance with technical standards.
  • Performed troubleshooting for failed software installs and hardware issues, escalating as necessary.

Hardware & Lifecycle Support

  • Performed basic hardware repairs (e.g., motherboard, screen, battery swaps) and refurbishments for reuse.
  • Participated in inventory tracking and device lifecycle management from acquisition to retirement.

Web Development Project

  • Improved the internal React-based website used for tracking hardware builds, returns, and inventory.
  • Connected frontend updates to backend Node.js endpoints and worked with SQL databases.
  • Added a model mismatch check feature that insured returns and build computers are being stored as the correct model, improving consistency and asset tracking.
  • Currently, I am working on a automatic return process feature which will automate the return process procedures. This includes a new UI modal connected to the backend via API calls.
  • Developed a calendar feature for the asset tracking website to track staff builds, returns, and days off using ReactJS and MSSQL.
